## âš¡ Performance Tips

1. **Lazy Load:** Load match lists only when section is visible
2. **Cache:** Store match data in state; refresh on action
3. **Pagination:** Limit initial load (e.g., 10 matches per section)
4. **Polling:** Refresh live section every 5-10 seconds
5. **Real-time:** Consider WebSocket for live updates
